Unit 3 Manual Test Checklist

Use this checklist to verify functionality, visual feedback, and responsiveness.

Global Checks (All Simulators)

  1. Load each simulator without console errors.
  2. Click every preset button and verify controls update.
  3. Adjust each slider and verify outputs recalculate immediately.
  4. Click Reset and verify default values/state are restored.
  5. Hover interactive controls and confirm visual feedback appears.
  6. Resize to mobile width and verify no overlap or clipping.

Energy Conversion Lab

  1. Check Potential, Kinetic, and Thermal values update with each input change.
  2. Verify bar widths reflect relative magnitudes.
  3. Verify drop-object position changes with drop height and heat haze intensifies with friction.
  4. Click Drop Object and verify animated fall + impact flash trigger.
  5. Verify impact velocity, efficiency, and conservation text update correctly.
  6. Confirm summary note changes between good, balanced, and warning states.

Orbital Motion Simulator

  1. Confirm circular velocity, escape velocity, and period all update live.
  2. Set high speed and verify status becomes escape trajectory.
  3. Set low speed and verify status becomes suborbital or re-entry.
  4. Verify orbit visualization switches behavior (stable/ellipse/escape/re-entry) with status.
  5. Verify ellipse height changes with speed difference (eccentricity behavior).
  6. Verify speed scale markers (v, vc, ve) move correctly.

Resonance Lab

  1. Verify amplitude ratio and frequency ratio update continuously.
  2. Set high damping and check response changes to heavily damped.
  3. Set drive frequency near natural frequency and check at resonance state.
  4. Verify oscillator spring/mass motion visibly changes with amplitude and damping.
  5. Verify phase label and driver indicator behavior update with ratio changes.
  6. Verify response curve current-point marker tracks current settings.
  7. At extreme resonance, verify cap warning appears.
